AI 预测流程

预测生成流程图
1 收集比赛上下文
球队排名、历史交锋、近期状态、伤停情况
↓
2 构建提示词
将上下文信息组织成结构化 prompt
↓
3 调用大模型
发送请求到 Claude/GPT API
↓
4 解析并缓存结果
提取比分、置信度、分析理由

AI Prompt 模板（示例）

你是专业的足球比赛预测分析师。请根据以下信息预测比赛结果：

**比赛信息**
- 巴西 (FIFA排名第3) vs 克罗地亚 (FIFA排名第12)
- 比赛时间：2026年6月18日 19:00
- 比赛场地：卢赛尔体育场

**球队状态**
- 巴西近5场：2-1胜、3-0胜、1-1平、2-0胜、4-1胜
- 克罗地亚近5场：1-0胜、0-1负、1-1平、2-1胜、1-1平

**历史交锋**
- 总计：巴西3胜1平1负
- 上次交手：2014年世界杯，巴西3-1克罗地亚

**伤停情况**
- 巴西：内马尔（伤疑）
- 克罗地亚：无重要伤停

请以JSON格式返回预测：
{
  "home_score": 预测主队进球数,
  "away_score": 预测客队进球数,
  "home_win_percent": 主队胜率,
  "draw_percent": 平局率,
  "away_win_percent": 客队胜率,
  "confidence": 整体置信度(0-100),
  "reasoning": "简短分析理由"
}
